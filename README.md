# CyberHole 赛博朋克黑洞入侵

一个参考抖音小游戏「黑洞入侵」(Hole.io) 核心玩法的网页游戏。单文件 HTML5 Canvas 实现，零依赖、零素材。

## 玩法

控制赛博朋克风格的黑洞，在城市中吞噬比自己小的物体来成长，与 AI 对手竞争。

- **限时赛**：2 分钟内尽可能吞噬，争夺最高分和排名
- **无尽模式**：撑到最后，AI 随你变强，达到最高级「城市杀手」即胜利

## 操控

- **WASD / 方向键 / 鼠标拖拽** 控制移动
- **ESC** 返回模式选择
- **Q** 主动结束看成绩（无尽模式）

## 特性

- 矢量霓虹美术 + 旋涡吸积盘黑洞
- 吞噬汁水感：螺旋坠入动画、屏幕震动、吸附粒子
- Web Audio API 程序化音效（吞噬、连击、死亡）
- 道具系统：⚡加速 / ◎磁铁 / ❖护盾
- 成就系统（LocalStorage 持久化）
- AI 攻击性：会主动追击吞噬比自己小的对手

## 技术

- 纯前端单文件，无构建步骤
- 精灵图缓存优化（预渲染矢量物体，每帧只 drawImage）
- 空间哈希碰撞检测

## 本地运行

直接用浏览器打开 `index.html` 即可。
